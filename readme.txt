== Description ==

Save a lot of time when to work in WordPress backend by powerful AJAX search and other useful tools.

= Features =

- Powerful AJAX search for post, page, post types, category, tag, taxonomies, user...
- Search product and order (Premium Version)
- Useful tools as widgets
- Front-end support
- WPML integration

= Translators =

Available Languages: English (Default)

If you have created your own language pack, or have an update for an existing one, you can send [gettext PO and MO file](http://codex.wordpress.org/Translating_WordPress "Translating WordPress") to [us](https://wpclever.net/contact?utm_source=pot&utm_medium=wpas&utm_campaign=wporg "WPClever.net") so we can bundle it into WPC Smart Search.

= Need support? =

Visit [plugin documentation website](https://wpclever.net?utm_source=doc&utm_medium=wpas&utm_campaign=wporg "plugin documentation").

== Installation ==

1. Go to Plugins in the Dashboard and select "Add New"
2. Search for "WPC Smart Search", Install & Activate it
3. Now you can access to WPC Smart Search by clicking into to right conner menu ( http://prnt.sc/dt7tx3 ) or keyboard shortcut Ctrl + M
